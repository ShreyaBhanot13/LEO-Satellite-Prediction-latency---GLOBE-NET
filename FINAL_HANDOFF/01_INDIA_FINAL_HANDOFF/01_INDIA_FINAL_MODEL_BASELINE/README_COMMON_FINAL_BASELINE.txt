COMMON FINAL BASELINE
========================================

This folder contains the official final baseline used by the current project.

SUBFOLDERS:
- DATASETS: final zone-risk dataset and feature list
- MODELS: final saved CatBoost zone-risk model
- TRAIN_TEST_SPLITS: saved train and test splits for reproducibility
- RESULTS: model comparison, predictions, confusion matrix, and state summary
- REPORTS: final dataset/modeling/tuning reports
- SCRIPTS: dataset build, model training, and tuning scripts

USE THIS FOLDER WHEN:
- You want the final dataset
- You want the final model
- You want the real final metrics
- You need the official baseline artifacts for report writing

DO NOT CONFUSE THIS WITH:
- older regression artifacts
- older latency-classifier artifacts
- operator-enriched comparison experiments
