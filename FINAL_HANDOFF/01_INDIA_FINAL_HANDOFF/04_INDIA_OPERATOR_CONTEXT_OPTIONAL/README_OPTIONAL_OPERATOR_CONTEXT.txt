OPTIONAL OPERATOR CONTEXT
========================================

These files are supplementary only.
They were used to test whether operator-based TRAI context improves the final model.

IMPORTANT:
- This is not the official final model package.
- The operator-enriched experiment did not beat the final baseline on balanced accuracy and high-risk F1.
- Use this folder only for additional telecom-context discussion in the report or viva.

INCLUDED:
- operator-enriched dataset
- operator-enriched feature list
- baseline vs operator comparison results
- operator comparison reports
- TRAI provider summaries
