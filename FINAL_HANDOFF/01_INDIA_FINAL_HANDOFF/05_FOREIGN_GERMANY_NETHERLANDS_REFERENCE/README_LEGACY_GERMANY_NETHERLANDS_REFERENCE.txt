LEGACY GERMANY / NETHERLANDS REFERENCE
========================================

This folder contains the older Europe-side benchmark assets that used Germany and Netherlands data.

WHAT IS INSIDE:
- Raw Germany iperf files
- Raw Netherlands iperf files
- The older engineered benchmark dataset
- Older Europe-benchmark model files and train/test splits
- Legacy benchmark reports and notes

IMPORTANT:
- This is not the official final India-first baseline model package.
- Use this folder only as reference material for the earlier global / Europe benchmark track.
- The current final project baseline is still the India zone-risk classifier in 01_COMMON_FINAL_BASELINE.

WHEN TO USE THIS FOLDER:
- If someone on the team wants the Germany/Netherlands datasets that were part of the older global story
- If you need to reference the older Europe benchmark model in the report
- If you want to show that the project also had an earlier Europe-compatible benchmark pipeline
